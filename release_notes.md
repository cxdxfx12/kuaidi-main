# 🚀 v2.0.0 重大性能升级

## 核心改进

### ⚡ Excel 读取提速 **5.3倍**
- **原方案**: openpyxl (纯 Python) 逐行解析
- **新方案**: python-calamine (Rust 引擎) 直接读取
- **实测**: 75MB 文件从 **117秒 → 22秒**

### 🎯 规则匹配引擎重写
- 活动加价规则预解析为 tuple 缓存，跳过运行时 JSON/字符串转换
- 每行匹配从 **~30µs → ~4.5µs**

### 📤 导出速度翻倍
- JSON 字段解析 → 预编译正则直接提取（少一次 json.loads）
- xlsxwriter `constant_memory=False`，写入快 **33%**

### 🗃️ SQLite 写入优化
- WAL 模式 + synchronous=OFF
- **50万行** 批量提交事务，减少磁盘 IO 同步次数
