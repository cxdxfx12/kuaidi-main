# 杭州喵喵至家网络有限公司 · 派费计算系统

## 一、项目概述

一个基于Web的**快递派费结算与财务管理系统**，支持导入Excel表格，自动计算派费、多级结算、月结对账，并导出结果。

- **公司名称**：杭州喵喵至家网络有限公司
- **项目类型**：内部工具（B方案 · Web端）
- **数据规模**：100万+条记录（大数据量场景）
- **当前阶段**：架构设计（待补充派费计算规则）

> ⚠️ **大数据量特别说明**：本系统需处理100万+条数据，架构设计已针对此场景优化，详见「九、大数据量处理方案」。

## 九、大数据量处理方案（100万+条数据）

### 9.1 Excel格式限制与解决方案

| 问题 | 说明 | 解决方案 |
|------|------|----------|
| **行数上限** | .xlsx格式上限104万行 | 改用CSV格式，或分Sheet/分文件 |
| **文件体积** | 100万行可能达到100MB+ | 后端流式读取，不要一次性加载 |
| **前端解析** | 浏览器直接解析会卡死 | 后端处理，前端只展示结果 |

**建议**：优先支持CSV上传，或后端自动将Excel转换为CSV处理。

### 9.2 内存管理策略

#### ❌ 错误做法（内存爆炸）
```python
# 一次性加载全部数据 - 100万行可能占用1GB+内存
df = pd.read_excel("large_file.xlsx")
```

#### ✅ 正确做法（分块处理）
```python
# 方案1：分块读取（推荐）
for chunk in pd.read_csv("large_file.csv", chunksize=10000):
    result = process_chunk(chunk)
    save_to_db(result)

# 方案2：流式处理
with pd.read_csv("large_file.csv", iterator=True) as reader:
    for chunk in reader.get_chunk(5000):
        process(chunk)
```

### 9.3 异步任务队列（Celery）

```python
# backend/app/tasks/calculate_task.py
from celery import Celery
from celery.result import AsyncResult

celery_app = Celery('fee_calculator', broker='redis://localhost:6379/0')

@celery_app.task(bind=True)
def calculate_fees_task(self, file_path, total_rows):
    """异步计算派费任务"""
    processed = 0
    results = []
    
    for chunk in pd.read_csv(file_path, chunksize=10000):
        # 计算当前块
        chunk_result = calculate_chunk(chunk)
        results.extend(chunk_result)
        
        # 更新进度
        processed += len(chunk)
        progress = int(processed / total_rows * 100)
        self.update_state(state='PROGRESS', meta={'progress': progress})
    
    # 批量写入数据库
    bulk_save_to_db(results)
    
    return {'status': 'success', 'total_rows': processed}
```

```python
from fastapi.responses import StreamingResponse

@app.get("/api/tasks/{task_id}/progress")
async def get_task_progress(task_id: str):
    async def event_stream():
        while True:
            result = AsyncResult(task_id)
            if result.state == 'PROGRESS':
                yield f"data: {json.dumps({'progress': result.info['progress']})}\n\n"
            elif result.state == 'SUCCESS':
                yield f"data: {json.dumps({'progress': 100, 'status': 'done'})}\n\n"
                break
            await asyncio.sleep(1)
    
    return StreamingResponse(event_stream(), media_type="text/event-stream")
```

```python
from celery import group, chord

@celery_app.task
def process_single_file(file_path, file_name):
    """处理单个文件"""
    # 分块计算
    results = []
    for chunk in pd.read_csv(file_path, chunksize=10000):
        results.extend(calculate_chunk(chunk))
    return {'file_name': file_name, 'total_rows': len(results), 'total_fee': sum_fee(results)}

@celery_app.task
def merge_batch_results(results_list):
    """汇总所有文件结果"""
    total_fee = sum(r['total_fee'] for r in results_list)
    total_rows = sum(r['total_rows'] for r in results_list)
    return {
        'files': results_list,
        'total_fee': total_fee,
        'total_rows': total_rows
    }

# 批量任务分发
def start_batch_calculation(file_list):
    """启动批量计算"""
    # 创建任务组（并行处理各文件）
    job = group(
        process_single_file.s(f['path'], f['name']) for f in file_list
    ) | merge_batch_results.s()
    
    result = job.apply_async()
    return result.id
```

```sql
CREATE TABLE fee_details (
    id BIGINT AUTO_INCREMENT,
    record_id INT,
    row_data JSON,
    calculated_fee DECIMAL(12,2),
    rule_applied VARCHAR(100),
    remark VARCHAR(255),
    created_at DATETIME,
    PRIMARY KEY (id, record_id)
) PARTITION BY RANGE (record_id) (
    PARTITION p1 VALUES LESS THAN (100),
    PARTITION p2 VALUES LESS THAN (200),
    PARTITION p3 VALUES LESS THAN (300),
    PARTITION p_max VALUES LESS THAN MAXVALUE
);
```
